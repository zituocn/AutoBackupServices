# AutoBackupServices

.net windows服务，固定时间拖取其他服务器上的数据库及文件最新备份 
使用.net 4.0开发，可部署在windows服务器上

## 安装

```sh
git clone https://github.com/zituocn/AutoBackupServices
```

使用Microsoft Visual Studio 2010或以上版本打开


## 配置

```
手动修改 `conf/config.json` 和`server.json`

```

## config.json


`localroot 为备份产生的目录` 
`hour 为开始备份的时间`
